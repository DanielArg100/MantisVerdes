# 🍏 Website Mantis Verdes 🟩

- Descripción<br>
✨ Sitio Web sobre la orden Mantis Verdes del manga Black Clover (No lei el manga ni vi el anime) ✨
    - 🌳 Pasate por la página web de Productos para ver todos los productos a la venta sobre la orden Mantis Verdes 📗
    - 🟢 Pasate por la página web de Galería para ver imagenes relacionadas ❎
- Colaborador/es:

## Lenguajes Utilizados
- HTML: FrontEnd, Estructura.
- CSS: FrontEnd Diseño.
- JavaScript: BackEnd, Lógica.

### Estructura
```
GOT
    css
        style.css       # --> Hoja de Estilos
    Pages
        Contactos.html  # --> Página Web Secundaria
        Galería.html    # --> Página Web Secundaria
        Nosotros.html   # --> Página Web Secundaria
        Productos.html  # --> Página Web Secundaria
    script 
        db.js           # --> Base de Datos (Arrays)
        main.js         # --> Lógica de la Página
    src                 # --> Archivos Estáticos
    index.html          # --> Página Base
```
